  1. a 태그 : 유일한 클릭이벤트

    + jQuery에서는 focus이벤트를 a태그가 아닌 다른 대상에게 부여할 수 있음

  1. a 태그
  2. 대상:focus

jQuery에서 사용할 수 있는 event 기능

1.click

대상을 클릭하기위해 사용하는 이벤트 기능

$('#clickEvent').click(function(){

$(this).css({'backgroundColor':'lightblue'}).fadeTd(1000,'0.3');

});
대상을 클릭하면 배경색상 변경

2.mouseover

대상에 마우스를 올렸을때 기능을 구현하기위해 사용하는 이벤트 기능

$('#mouseoverEvent').mouseover(function(){

$(this).animate({'color':'#f00'},500,liner);

});
대상에 mouseover하면 글자색상 변경

3.mouseout

대상에서 마우스를 뗐을때 기능을 구현하기위해 사용하는 이벤트 기능

$('#mouseoutEvent').mouseout(function(){

$(this).animate({'borderRadius':'50%'},500,ease);

});
대상에 mouseout하면 border-radius 변경

4.mouseenter

대상에 마우스를 올렸을때 기능을 구현하기위해 사용하는 이벤트 기능

$('#mouseenterEvent').mouseenter(function(){

$(this).animate({'color':'#f00'},500,liner);

5.mouseleave

대상에서 마우스를 뗐을때 기능을 구현하기위해 사용하는 이벤트 기능

$('#mouseleaveEvent').mouseleave(function(){

$(this).animate({'borderRadius':'50%'},500,ease);

});
대상에 mouseleave하면 border-radius 변경

mouseenter 과 mouseover 메서드의 차이

mouseover는 직접 이벤트를 걸지않은 자식요소에 마우스 포인터가 와도 발생하지만, mouseenter 이벤트는 문서 객체의 안에 있는지 외부에 있는지만 따지기 때문에 이벤트가 한 번만 발생하게 됩니다.

html
<div class="outer">
<div class="inner"></div>
</div>

css
* {margin:0; padding:0;}
a:link, a:visited {text-decoration:none; color:#fff;}
a:hover, a:focus {color:#f00; text-decoration:underline;}
li {list-style:none;}
.wrapper {width:960px; margin:0 auto;}

.outer {
width: 200px;
height: 200px;
background: Orange;
padding: 50px;
margin: 10px;
}

.inner {
width: 100%;
height: 100%;
background: Pink;
}

script
$('.outer')
.mouseover(function(){
$('.dif_desc').append('

마우스오버가 발생했습니다.

')
})
.mouseenter(function(){
$('.dif_desc').append('

마우스엔터가 발생했습니다.

')
});

mouseout 과 mouseleave 메서드의 차이

mouseout는 직접 이벤트를 걸지않은 자식요소에 마우스 포인터가 와도 발생하지만, mouseleave 이벤트는 문서 객체의 안에 있는지 외부에 있는지만 따지기 때문에 이벤트가 한 번만 발생하게 됩니다.

focus 과 focusin 메서드의 차이

focusin, focusout는 직접 이벤트를 걸지않은 자식(또는 부모)요소에 tab이 와도 발생하지만(버블링 현상), focus, blur 이벤트는 문서 객체의 안에 있는지 외부에 있는지만 따지기 때문에 이벤트가 한 번만 발생하게 됩니다.(버블링 발생하지 않음)
  • 1. 동일한 결과를 예상하고 싶을때 focusin과 focusout를 함께, focus와 blur를 함께 사용하자.
  • 2. input에 포커스되거나 잃어버렸을때, 부모요소에서 리스너를 동작하고 싶다면 focusin과 focusout을 사용하자.

html
<div class="parent">
<input type="text">
</div>

script
$('.parent').focusin(function(){
  console.log('부모 focusin!');
});
$('input').focusin(function(){
  console.log('인풋 focusin!');
});
$('.parent').focus(function(){
  console.log('부모 focus!');
});
$('input').focus(function(){
  console.log('인풋 focus!');
});

버블링 현상

이벤트 버블링은 보통 선택한 엘리먼트가 속해있는 부모요소에 의하여 발생한다.
예를들어 어떤 버튼을 클릭하려고 할때 그 버튼을 가지고 있는 부모 요소에 이미 클릭 이벤트가 존재할때,
선택한 요소와 더불어 의도하지 않았던 부모 요소의 이벤트도 함께 실행되는 문제점을 발생시킬 수 있을 것이다.

html
<div class="dep1" onclick="doDep1(event);">
  dep1
  <div class="dep2" onclick="doDep2(event);">
     dep2
     <button onclick="doDep3(event);">dep3</button>
  </div>
</div>

css
.dep1 {
width: 200px;
height: 200px;
display: block;
background-color: #dcdcdc;
}
.dep2 {
width: 100px;
height: 100px;
display: block;
margin: 3px auto;
background-color: #ececec;
}
button {
display: block;
margin: 0 auto;
}

script
<script type="text/javascript">
var doDep1 = function(event) {
   alert('Clicked dep1');
}
var doDep2 = function(event) {
   alert('Clicked dep2');
}
var doDep3 = function(event) {
  ('Clicked dep3');
}
<s/script>

dep2

stopPropagation() 함수를 사용하면 부모에 의한 이벤트 버블링을 효과적으로 매우 간단하게 막을 수 있게된다.

html
<div class="dep1" onclick="doDep4(event);">
  dep1
  <div class="dep2" onclick="doDep5(event);">
     dep2

  </div>
</div>

var doDep4 = function(event) {

}

event.stopPropagation();
   event.stopPropagation();
  ('Clicked dep3');
}

javascript에서 사용할 수 있는 마우스 event 기능

html
<h3>onclick : 클릭할때 생기는 이벤트</h3>
<input id="clickBtn" type="button" value="클릭" onclick="onClickBtn();"/>

<h3>ondblclick : 더블클릭 할때 생기는 이벤트</h3>
<input id="dblBtn" type="button" value="더블클릭" ondblclick="onDblBtn();"/>

<h3>onmousemove : 마우스가 움직일때 생기는 이벤트</h3>
<input id="mouseMov" type="button" value="마우스 움직이기" onmousemove="onmouseMov();"/>

<h3>mouseout : 마우스가 바깥으로 나가면 생기는 이벤트</h3>
<input id="mouseOut" type="button" value="마우스 아웃" onmouseout="onmouseOut();"/>

<h3>onmouseup : 1) 해당하는 값 누르기, 2) 해당하는 값에서 마우스를 놓을때 생기는 이벤트 </h3>
<input id="mouseUp" type="button" value="마우스 업" onmouseup="onmouseUp();"/>

<h3>onmouseDown : 마우스 버튼을 누를 때 발생</h3>
<input id="mouseUp" type="button" value="마우스 다운" onmousedown="onMsDown();"/>

script
<script>
function onClickBtn(){ alert("클릭 !"); }
function onDblBtn(){ alert("더블클릭 !"); }
function onmouseMov(){ alert("마우스 움직이기 !"); }
function onmouseOut(){ alert("마우스 아웃 !"); }
function onmouseUp(){ alert("마우스 업 !");}
function onMsDown(){alert("마우스 다운 !");}
</script>

onclick : 클릭할때 생기는 이벤트

ondblclick : 더블클릭 할때 생기는 이벤트

onmousemove : 마우스가 움직일때 생기는 이벤트

mouseout : 마우스가 바깥으로 나가면 생기는 이벤트

onmouseup : 1) 해당하는 값 누르기, 2) 해당하는 값에서 마우스를 놓을때 생기는 이벤트

onmouseDown : 마우스 버튼을 누를 때 발생

1. 두가지 기능을 한꺼번에 묶을때 사용하는 기능
2. 접근성을 고려한 마우스기능과 키보드 기능을 편리하게 묶어서 구현할 수 있다.
3. 종류 : bind, on, one

bind

  • bind는 정적인 페이지에서 기능을 묶을때 사용
  • 해제속성 : unbind (바인드 끊기)
  • 표준 브라우저에서 주로 사용 (jQuery 3.0버전에서는 사용되지 않는다.)

html
<div class="css_test">
<img src="http://superkts.com/img/css/dog060.gif" />
<img src="http://superkts.com/img/css/dog060.gif" />
<img src="http://superkts.com/img/css/dog060.gif" />
<img src="http://superkts.com/img/css/dog060.gif" />
<img src="http://superkts.com/img/css/dog060.gif" />
</div>

css
.css_test {
  border-radius : 5px;
  border : 3px solid gray;
  padding : 20px;
  text-align : center;
}
.css_test img {
  cursor : pointer;
}

script
<script type="text/javascript">
  $('.css_test img').bind({
    mouseover : function(){
      this.src = 'http://superkts.com/img/css/dog061.gif';
    },
    mouseout : function(){
      this.src = 'http://superkts.com/img/css/dog060.gif';
    },
    click : function(){
      $(this).animate({top:'-=70'}, 100, function(){
        $(this).animate({top:'+=70'}, 100, function(){
          this.src = 'http://superkts.com/img/css/dog021.gif';
        });
      });
    }
  });
</script>

3개의 이벤트 동시 바인딩
마우스 오버, 아웃, 클릭

해제하는 방법
unbind를 사용해 빼고싶은 이벤트기능을 해제할 수 있다.
$('.css_test img').unbind('click');

on

  • on()은 jQuery 1.7부터 bind(), delegate(), live()를 대체하는 메서드로 이벤트 핸들러 바인딩에 필요한 기존 메서드의 모든 기능을 제공한다. jQuery는 앞의 세 메서드 대신 on()을 사용할 것을 권장하고 있다.
  • 구형, 표준 브라우저에서 사용할 수 있다.
  • 해제속성 : off

.on( events [, selector] [, data], handler( eventObject ) )

events: 공백으로 구분된 하나 이상의 이벤트 타입과 옵션인 네임스페이스. "click", "keydown.myPlugin", ".myPlugin" 등이 있음
selector: 이벤트가 발생할 요소들의 자손을 찾는 선택자. 선택자가 null 이거나 생략됐다면 이벤트는 선택된 요소에 한해서 반응한다.
data: 이벤트가 발생할 때 핸들러에 전달할 데이터
handler(eventObject): 이벤트가 발생되면 실행될 기능. false를 반환하는 함수라면 간단하게 false 를 직접 인자로 하면 된다.

html
<div id="demo8">
    <button>눌러보세요</button>
</div>

script
$("#demo8 button").on("click mouseout", function() {
    alert("이벤트 발생!");
});
