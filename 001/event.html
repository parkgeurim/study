마우스와 키보드 EVENT

xhtml 에서 사용할 수 있는 event 기능

  1. a 태그 : 유일한 클릭이벤트
  2. 대상:focus 마우스 기능을 대신하여 tab키로 접근 가능하도록 만드는 기능
    + a태그가 있으면 focus 접근이 가능
    + jQuery에서는 focus이벤트를 a태그가 아닌 다른 대상에게 부여할 수 있음
  3. 탭키로 작동할 수 있는 skip navigation

html5에서 사용할 수 있는 event 기능

  1. a 태그
  2. 대상:focus
  3. 대상:hover
  4. 이벤트의 기능이 다양하지 못하기 때문에 jQuery, javacript의 기능들을 활용한다.

jQuery에서 사용할 수 있는 event 기능

1.click

$('#clickEvent').click(function(){

$(this).css({'backgroundColor':'lightblue'}).fadeTd(1000,'0.3');

});
대상을 클릭하면 배경색상 변경

2.mouseover

$('#mouseoverEvent').mouseover(function(){

$(this).animate({'color':'#f00'},500,liner);

});
대상에 mouseover하면 글자색상 변경

2.mouseenter

$('#mouseenterEvent').mouseenter(function(){

$(this).animate({'color':'#f00'},500,liner);

});
대상에 mouseenter하면 글자색상 변경

3.mouseout

$('#mouseoutEvent').mouseout(function(){

$(this).animate({'borderRadius':'50%'},500,ease);

});
대상에 mouseout하면 border-radius 변경

3.mouseleave

$('#mouseleaveEvent').mouseleave(function(){

$(this).animate({'borderRadius':'50%'},500,ease);

});
대상에 mouseleave하면 border-radius 변경

3.mouseout

$('#mouseoutEvent').mouseout(function(){

$(this).animate({'borderRadius':'50%'},500,ease);

});
대상에 mouseout하면 border-radius 변경

4.dblclick

$('#dblclickEvent').dblclick(function(){

alert('더블클릭!');

});
대상에 더블클릭하면 alert창 만들기

5.focus

$('#focusEvent').focus(function(){

alert('focus!');

});
대상에 focus하면 alert창 만들기

6.blur

$('#blurEvent').blur(function(){

alert('focus!');

});
대상에 blur(포커스가 흐려짐)하면 alert창 만들기

7.focusin

$('#focusinEvent').focusin(function(){

alert('focusin!');

});
대상에 focusin하면 alert창 만들기

7.focusout

$('#focusoutEvent').focusin(function(){

alert('focusout!');

});
대상에 focusout하면 alert창 만들기
